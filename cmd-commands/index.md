# Commands

## CMD commands

### Close and open explorer

- Opening

```batch
 start explorer.exe
```

- Closing (kill)

```batch
 taskkill /f /im explorer.exe
```

---

## Bash commands

### Open file

```bash
explorer index.html
```

### Create file

```batch
touch yourfile.ext
```

### Delete file

```batch
rm filename.ext
```

> Use <kbd>-f</kbd> argument to forcefully delete a file.
> Use <kbd>-r</kbd> argument to recursively delete along with its content.

### Create folder/s

```batch
mkdir foldername
```

> Use <kbd>-p</kbd> argument to create nested directories.

### clear console

```batch
cls
```
