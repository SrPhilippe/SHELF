## Git commands

* **`git init`**
inits a new repository

* **`git add`**
* > adds a file to the commits files
    - ### `git add .`
    - adds all the files directory to the commits files

## `git status`
> shows the status repository

* ## `git commit`
* > commit added files
    - ### `git commit -m "blah blah blah"`
    - > commits with the text "blah blah blah"
